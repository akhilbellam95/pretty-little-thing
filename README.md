# prettyLittleThing

Steps to run this project:

## Install dependencies:

```bash
$ yarn
```

## Run

```bash
# run in development mode
$ yarn start
```

### Build for Android

1. Build the Android app:

```bash
$ yarn android
```

### Build for iOS

1. Build the iOS app:

```bash
$ yarn ios
```

## Test

```bash
# unit tests
$ yarn test

# updates test snapshots
$ yarn test:update-snapshot

# generates test coverage
$ yarn test:coverage
```
